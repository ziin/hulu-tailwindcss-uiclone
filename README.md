## Getting Started

You must place your own [TMDB API KEY](https://www.themoviedb.org/) into `.env.local`.  
Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
